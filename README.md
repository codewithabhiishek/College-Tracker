# College Tracker

A React application for tracking college applications.

## Prerequisites

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the app: `npm run dev`

## Features

- User Authentication (Mocked)
- Dashboard for tracking university application statuses
- Document management

## Tech Stack

- React
- Vite
- Tailwind CSS
- Radix UI Components
- React Router
